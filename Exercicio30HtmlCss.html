Exemplo 30 - HTML CSS

BOX MODEL CSS

Box model é um conceito fundamental que trata cada elemento HTML como uma caixa, um retangulo. Essa Caixa retangular possui propriedades que definem o posicionamento do elemento HTML na pagina.

O Box model possui tres propriedades fundamentais: Border, margin e padding.

  • Border: A propriedade border é destinada a estilização da borda do componente HTML, ou seja, ela circunda o preenchimento do conteudo.
  • Padding: A propriedade Padding refere-se ao preenchimento da box model, ele é transparente e cria uma area livre ao redor do conteudo, que é onde aparecem textos e imagens.
  • Margin: A propriedade Margin refere-se ao espaço ou area apos a borda do componente HTML.

Margin e Padding

Quando trabalhamos com as propriedades margin e padding podemos definir os valores de espaçamento externo, após a borda, ou interno(padding), para isso definimos valores para cima(Top), Direita(Right), a baixo (Bottom) e a esquerda(Left).

Em todas situações podemos usar a regra do relógio e espelho.

Regra do relógio

A regra do relógio é uma forma rápiuda e organizada de definir propriedades de quatro lados do box model, usando apenas uma linha de codigo CSS

Regra do espelho(comportamento de valores)

A regra do espelho não é um termo técnico oficial, mas sim uma forma didática de entender como o CSS lida com afalta de valores na regra do relógio para manter a simetria. Neste exemplo, estamos aplicando as diversas propriedades de margin e padding.
