CSS

O CSS é o acrônimo de folhas de estilo em cascata, ele é usado para estilizar documentos HTML. Ele deve ser aplicado as tags HTML, usado para formatações diversas do documento, tais como: mudanças de cores, fontes e posicionamentos de elementos HTML.

Tipos de seletores - seletores Tipo

Os seletores de tipo são aqueles que representam tags html, eles são utilizados para aplicar as regras CSS neste exemplo estamos usando os seletores de tipo. Analise a pagina.
