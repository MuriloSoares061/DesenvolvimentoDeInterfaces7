Exemplo 7 - HTML - listas ordenadas e não ordenadas HTML

Este exemplo mostra como criar listas ordenadas e não ordenadas, incluindo subitens(listas alinhadas).

Descrição das tags utilizadas na criação das listas.

  • <ol>:Define uma lista ordenada(com números, letras e ou algarismos romanos)
  • <ul>:Define uma lista não ordenada (com marcadores como pontos e quadrados)
  • <li>:Define um elemento na lista
