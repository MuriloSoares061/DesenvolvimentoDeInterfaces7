Exemplo 27 HTML CSS - Vinculação interna (inline)

A Vinculação Interna CSS é um metodo para aplicar estilos a uma documento html, inserindo um codigo css Diretamente na pagina(Incorporado) ou como componente html(inline)

Esse estilo de vinculação não é muito recomendado pois dificulta a manutenção da pagina, uma vez que se tivermos varios componetes e varias paginas devemos entrar em cada uma das paginas para fazer as alterações.

Vinculação Inline

A vinculação inline é a forma de aplicar estilos diretamente a um elemento HTML, usando o atributo style na tag de abertura, Essa abordagem vincula o estilo exclusivamente ao elemento que estiver sendo criado. É importante dizer que essa abordagem não deve ser usada de forma comum. pois dificulta a manutenção da pagina.
