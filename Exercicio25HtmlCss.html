Exemplo 25 - HTML CSS - Vinculação de Arquivos

Vinculaçao de arquivos CSS

A Vinculção de arquivos CSS é o processo de conectar uma folha de estilos a uma pagina HTML para que os elementos visuais da pagina recebam formatação.

Existem Basicamente duas formas de vinculação de arquivos CSS em documentos HTML: a Vinculação externa e a Vinculação interna sendo que a forma mais recomendada será a Externa pois facilita a manutenção e o reuso das regras CSS.

Vinculação Externa

A Vinculação Externa como dito anteriormente é a mais recomendada em projetos reais. Nesse tipo de vinculação o CSS fica em um Arquivo separado e o HTML faz a ligação entre a tag por meio da tag link. Neste Documento estamos usando a vinculação Externa

Caracteristicas da Vinculação Externa

  • Separa a Estrutura html e a apresentação (regras Css)
  • Facilita a manutenção
  • Permite reutilizar o mesmo css em varias paginas
  • Deixa o projeto muito mais organizado

Vinculação Interna

A Vinculação Interna de arquivos CSS em documentos HTML se da de duas formas (Incorporada e Inline), Essa Vinculação ocorre diretamente no documento HTML por meio do Atributo STYLE. Ela deve ser utilizada com cuidado, pois Dificulta a manutenção e polui o HTML uma vez que mistura o conteudo com a apresentação.
