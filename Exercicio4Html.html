Exemplo 4 HTML - Estrutura Geral

Este Exemplo mostra a estrutura minima de um documento HTML, a seguir são escritas as principais tags utilizadas nessa estrutura Geral

  • <DOCTYPE html>: É uma instrução html, essa tag define o tipe do documento.
  • <html>: É o elemento central do documento, é o container do documento html, ele engloba toda a pagina.
  • <head>: Essa tag contem os metadados que serão interpretados peolo navegador, tals como titulos e metatags de codificação
  • <body>: Esse bloco é responsavel por deixar visivel o conteudo destinado aos usuario. Ou seja, todo o conteudo do documento html.
