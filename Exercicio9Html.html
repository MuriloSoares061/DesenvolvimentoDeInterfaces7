Exemplo 9 - HTML uso das tags link e tabelas

Este exemplo mostra como criar tabelas e links para permitir a navegação entre arquivos. As tags abordadas são:

  • <a>: Define um hyperlink
  • <table>: Define uma tabela
  • <tr>: Define uma linha da tabela
  • <th>: Define um cabeçalho para a tabela
  • <td>: Define uma célula da tabela
