Tags de seção/Títulos <h1> a <h6>

As tags de seção/título são utilizadas em documentos html para separar de forma hierarquica contúdos. Elas definem os títulos e subtitulos do documento html.

As tags utilizadas para definição dos títulos são <h1> a <h6> sendo que quanto menor for o número que vem junto ao h, maior será a sua importância, abaixo exemplificamos cada uma delas.
