Exemplo 21 HTML CSS

Seletor de Classe

O seletor de classe CSS, indicado por um ponto (.) seguido do nome da classe (ex: .minha-classe), seleciona elementos HTML com base no atributo class. Ele permite estilizar múltiplos elementos reutilizáveis com o mesmo estilo, oferecendo flexibilidade sem a especificidade alta de um ID. Neste exemplo estamos usando seletores de classe.
