Exemplo 31 HTML CSS

A borda em css representa uma linha visivel que envolve o conteúdo e o preenchimento, e é representada pela propriedade css border que é uma atalho mais simples, podendo o desenvolvedor utilizar aspectos mais individualizados dessa propriedade, tais como: border-style, border-width, border-color etc.

Neste exemplo, vamos mostrar os diferentes tipos de bordas que podem ser aplicados aos elementos html.
